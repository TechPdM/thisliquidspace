## thisliquidspace

Experimental homepage / POC using the Pretext text layout engine for browser-like typographic layout without DOM.

Vanilla TypeScript + Bun. Keep it simple and flexible — this is exploratory.

See `../pretext/CLAUDE.md` for the Pretext library API, implementation notes, and commands.

### Commands

- `bun start` — dev server at http://localhost:3001 (watch mode)
- `bun run build` — bundle TS/CSS into `dist/` for deployment
- `npx netlify deploy --dir=dist --prod` — deploy to Netlify

### Deployment

Hosted on Netlify: https://dulcet-paprenjak-fea3c7.netlify.app

Build outputs static HTML + bundled JS/CSS to `dist/`. Clean URLs handled via `_redirects` (generated by the build script).

### Important files

- `src/main.ts` — ripple reveal effect: DOM text color transitions + canvas scramble overlay
- `src/styles.css` — global styles
- `scripts/build.ts` — static site build script
- `index.html`, `projects.html`, `blog.html`, `about.html` — page content
